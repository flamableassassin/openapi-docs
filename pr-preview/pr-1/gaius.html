apiKey

Use the createapikey scope scope command in your server to get the api key for your server.

Available scopes:

  • warnings
  • caselogs
  • wordlists
  • notes
  • info
  • modlogs
  • leveling
Security Scheme Type: API Key
Header parameter name: api-key
